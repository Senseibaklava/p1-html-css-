Over onze opleiding

Soorten opleiding over Software developer

Software developer

Apps, website en zelfs de coolste games. Als software developer ga je aan de slag met het ontwerpen en bouwen van software. Je zorgt ervoor dat ze technisch goed werken. Je werkt vaak samen met collega’s én met de klant. Je luistert goed naar hun wensen en ideeën. En je gaat ermee aan de slag.

Software developer - Creative media technology

Je ontwerpt en bouwt software. Je bent creatief en zorgt voor een toffe applicatie – die er ook mooi uitziet. Jij zorgt er ook voor dat de voorkant van de app of website perfect werkt met de technologie op de achtergrond.

Wat is het opbouw van de opleiding?

Software developer

In 7 weken.

Week 1 :In de eerste week ga je alle softwares downloaden die nodig is om te kunnen beginnen met html/css.
Week 2 : Je gaat een begin maken over html en daarbij maak je een opdracht elke donderdag.
Week 3 : Je gaat een begin maken met css en je krijgt daarvoor een een opdracht waar je een tabel moet maken over de stand in het erendivisie.
Week 4 : Je gaat beginnen met je project voor een ouderavond waar je een website moet gaan bouwnen met 8 pagina's.
Week 5 : Je maakt een schets met figma over hoe je het wilt opbouwen.
Week 6 : Je maakt een begin met je project.
Week 7 : Je bent klaar met je website en je ga je voorbereiden voor de ouderavond waar je een presentatie moet gaan presenteren over een onderwerp over school.

Welke software gebruiken wij/Wat is Visual studio code?

Wij gebruiken visual studio code.
Visual Studio Code is een broncode-editor ontwikkeld door Microsoft voor Windows, Linux en macOS. Het bevat ondersteuning voor debugging, ingebouwde Git-controle, syntax highlighting, intelligente code-aanvulling, codefragmenten en code refactoring.

Wanneer is de ouderavond?/Wat wordt er besproken die avond?

Bij het ouderavond wordt er gesproken over je resultaten dit periode en over hoe je gedraagt tijdens de lessen en wordt er ook gekeken naar je aanwezigheid/afwezigheid.

Het ouderavond zal geplannend worden in de maand november.
